РУКОВОДСТВО НОВИЧКА ПО ЯЗЫКУ .DC
================================

Привет! Это твой путеводитель по языку программирования DC.
Здесь всё просто: мы пишем команды на русском языке, а компьютер их выполняет.

1. ОСНОВА ПРОГРАММЫ
   Каждая программа начинается с точки входа `подключение:` и заканчивается `отключение:0;`.
   
   #библиотека <ntstart.v>  // Это нужно, чтобы команда "да" работала
   
   подключение:
       да("Привет, мир!");  // Эта команда выводит текст в консоль
   отключение:0;
   }

2. ПЕРЕМЕННЫЕ (Где хранить информацию)
   Ты можешь сохранять числа или текст, чтобы использовать их позже.
   
   цел возраст = 25;         // Целое число
   плав рост = 1.85;         // Дробное число
   
   да("Мне лет:");
   да(возраст);

3. УСЛОВИЯ (Если... то...)
   Программа может принимать решения.
   
   если (возраст > 18) {
       да("Доступ разрешен");
   } иначе {
       да("Доступ запрещен");
   }

4. ЦИКЛЫ (Повторение действий)
   Чтобы повторить команду несколько раз.
   
   цел счетчик = 5;
   пока (счетчик > 0) {
       да(счетчик);
       счетчик = счетчик - 1; // Уменьшаем счетчик на 1
   }
   да("Время вышло!");

5. ПОЛЕЗНЫЕ БИБЛИОТЕКИ
   Библиотеки добавляют новые команды. Подключай их в самом начале.
   
   #библиотека <stat.v>     // Добавляет математику (сумма, среднее)
   
   Пример математики:
   цел итог = сумма(10, 20, 30);
   да(итог); // Выведет 60

6. ГЛАВНЫЕ СОВЕТЫ
   - В конце почти каждой команды ставь точку с запятой `;`.
   - Если программа не работает, посмотри на ошибки в черной консоли внизу.
   - Используй меню "Сборка -> Скомпилировать в EXE", чтобы создать настоящую программу (.exe файл).
